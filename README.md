# RetrAI Po 🚀

Mobile AI Assistant with streaming responses

## Features
- ⚡ ChatGPT-style streaming AI
- 📱 Mobile-first UI
- 🧠 Multi-model (Groq / OpenRouter)
- 🌐 PWA install support

## Deploy

1. Upload to GitHub (`retrai_po`)
2. Import to Vercel
3. Add ENV variables:

GROQ_API_KEY=
OPENROUTER_API_KEY=

Done 🚀
